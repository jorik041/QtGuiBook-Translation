Prefacio

Qt es un amplio framework en C++ para la creación de aplicaciones multiplataforma usando el enfoque de "escribir una vez, compilar en cualquier lugar". Qt permite a los programadores utilizar una única rama de código para aplicaciones que se ejecutarán en Windows 98 hasta Vista, Mac OS X, Linux, Solaris, HP-UX, y muchas otras versiones de Unix con X11. Las bibliotecas y herramientas de Qt también son parte de Qt/Embedded Linux, un producto que proporciona su propio sistema de ventanas sobre Linux embebido.

El propósito de este libro es enseñarle cómo escribir programas con interfaz gráfica usando Qt 4. El libro comienza con "Hola Qt", y progresa rápidamente hacia temas más avanzados, tales como la creación de widgets personalizados y el uso de drag & drop. El texto se complementa con un conjunto de ejemplos que se pueden descargar desde el sitio web del libro, http://www.informit.com/title/0132354160 . El Apéndice A explica cómo descargar e instalar el software, incluyendo un compilador C++ para aquellos que usan Windows.

Este libro está dividido en tres partes: La Parte I comprende todos los conceptos fundamentales y las prácticas necesarias para la programación de aplicaciones GUI con Qt. Asimilar esta parte es suficiente para escribir aplicaciones gráficas útiles. La Parte II cubre los temas centrales de Qt con mayor profundidad, y la Parte III dispone de material más especializado y avanzado. Usted puede leer los capítulos de las partes II y III, en cualquier orden,pero se asume que estará familiarizado con los contenidos de la Parte I. El libro también incluye varios anexos, como el Apéndice B que muestra cómo crear aplicaciones Qt y el Apéndice C, una introducción a Qt Jambi, la versión Java de Qt.

La primera edición con Qt 4 de este libro fue escrita sobre Qt 3, aunque fue revisada por completo para reflejar buenas prácticas extrapolables a Qt 4 e incluye nuevos capítulos sobre el modelo/arquitectura de Qt 4, el nuevo framework para plugins, programación integrada con Qt/Embedded Linux, y un nuevo apéndice. Esta segunda edición revisada y ampliada ha sido completamente actualizada para aprovechar las características introducidas en las versiones de Qt 4.2 y 4.3, e incluye nuevos capítulos sobre look & feel y scripting, así como dos nuevos apéndices. El capítulo original de gráficos se ha dividido en diferentes capítulos de 2D y 3D, que entre ellas ahora cubren las nuevas clases visoras de gráficos y el funcionamiento interno sobre OpenGL de QPainter. Además, ha sido añadido mucho material nuevo a los temas de bases de datos, XML, y los capítulos de programación integrada.

Esta edición, al igual que sus predecesoras, hace hincapié en la explicación de la programación en Qt y proporciona ejemplos reales, en lugar de simplemente un refrito o un resumen de la amplia documentación en línea de Qt. Debido a que el libro enseña sólidos principios y prácticas de programación Qt 4, los lectores serán capaces de aprender a usar los nuevos módulos que saldrán más adelante en las versiones de Qt 4.4, Qt 4.5 y Qt 4.X. Si usted está usando una de estas versiones modernas, asegúrese de leer la sección "Qué hay de nuevo en Qt 4.X" en la documentación de referencia para obtener una visión general de las nuevas características que están disponibles.

Hemos escrito el libro con el supuesto de que usted tiene un conocimiento básico de C++, Java o C#. Los ejemplos de código utilizan un subconjunto de C++, evitando muchas de las características de C++ que rara vez son necesarias cuando se programa usando Qt. En los pocos lugares donde es inevitable alguna construcción más avanzada de C++, se explica como se utiliza. Si ya sabe Java o C#, pero tiene poca o ninguna experiencia con C++, le recomendamos que primero lea el Apéndice D, que proporciona suficiente introducción a C++ como para ser capaz de utilizar este libro. Para una introducción más completa a la programación orientada a objetos en C++, le recomendamos "C++ How to Program" por P.J. Deitel y H.M. Deitel (Prentice Hall, 2007), y "C + + Primer" por Stanley B. Lippman, Josée Lajoie, y Barbara E. Moo (Addison-Wesley, 2005).

Qt se hizo su reputación como un framework multiplataforma, pero gracias a su API intuitiva y potente, muchas organizaciones utilizan Qt para el desarrollo sobre una sola plataforma. Adobe Photoshop Album es sólo un ejemplo de una aplicación para masas y para Windows escrita en Qt. Muchos sistemas de software sofisticados en mercados verticales, tales como herramientas de animación 3D, procesamiento de películas digitales, automatización de diseño electrónico (para el diseño de chips), la exploración de petróleo y gas, servicios financieros, y la imagen médica, están construidas con Qt. Si usted está teniendo éxito con un producto para Windows, usted puede crear fácilmente nuevos mercados en los mundos de Mac OS X y Linux simplemente recompilando.

Qt está disponible bajo diversas licencias. Si usted quiere construir aplicaciones comerciales, usted debe comprar una licencia comercial de Qt de Trolltech; si usted quiere construir programas de código abierto, puede utilizar la edición de código abierto (GPL). El entorno de escritorio K (KDE) y la mayoría de las aplicaciones de código abierto que lo acompañan se basan en Qt.

Además de los cientos de clases de Qt, existen complementos que amplían el alcance y el poder de Qt. Algunos de estos productos, al igual que los componentes de Qt Solutions, están disponibles en Trolltech, mientras que otros son ofrecidos por otras empresas y por la comunidad de código abierto, ver http://www.trolltech.com/products/qt/3rdparty/ para una lista de los add-ons disponibles. Los desarrolladores de Trolltech también tienen su propio sitio web, Trolltech Labs ( http://labs.trolltech.com/ ), en el que colocan código no-oficial que han escrito, porque es divertido, interesante o útil. Qt tiene una comunidad de usuarios bien establecida y próspera que utiliza la lista de correo cqt-interest , ver http://lists.trolltech.com/ para más detalles.
